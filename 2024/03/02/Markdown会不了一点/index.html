Markdown会不了一点

发表于2024-03-02|更新于2024-03-10|self study

1.标题

语法格式： '#'+'空格'+'文本'

注：

  • #和标题之间建议保留一个字符的空格
  • 一般情况下，文章标题会被系统渲染为一级标题，避免在文章中使用一级标题

2.列表

无序列表：'-'+空格+文本

有序列表：'1.'+空格+文本

3.链接和图片

链接：[显示文本]+(链接地址)

图片：'!'+[显示文本]+(链接地址)

  • 如果希望图片能够部署在网页上
    • 方法1：利用hexo-asset插件，使用如下格式：
    • {% asset_img xxx.jpg 图片名后可写图片描述 %}
    • 方法2：上传到github图床上，将图片本地地址转成网络地址（当下使用）

8.表格

Markdown制作表格使用|来分隔不同的单元格，使用-来分隔表头和其他行。

语法格式如下：

| 表头 | 表头 |

| — | — |

| 单元格 | 单元格 |

| 单元格 | 单元格 |

  • 自动生成太香啦😍
